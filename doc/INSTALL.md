# classify-my-dataset

A graphic user interface program to work with labels and images in a classification dataset.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/classify_my_dataset/

```bash
pip install --upgrade classify_my_dataset
```

Using:

```bash
classify-my-dataset
```

## Install from source
Installing `classify-my-dataset` program

```bash
git clone https://github.com/trucomanx-desktop/ClassifyMyDataset.git
cd ClassifyMyDataset
pip install -r requirements.txt
cd src
python -m build
pip install dist/classify_my_dataset-*.tar.gz
```
Using:

```bash
classify-my-dataset
```

## Uninstall

```bash
pip uninstall classify_my_dataset
```
